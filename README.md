# finance-analytics-sql-case-studies
This repository contains SQL-based analytical case studies focused on Finance and Revenue Operations.

The objective of these projects is to demonstrate how transactional datasets can be analysed using SQL to generate insights related to:

• Accounts Receivable efficiency and Working Capital behaviour  
• Customer payment risk and behavioural segmentation  
• Sales performance, marketing efficiency, and revenue quality  

Each case study includes:
- Business problem definition
- SQL-driven analytical approach
- Derived metrics and interpretation

Tools Used:
SQL (MySQL), Excel-based datasets, ERP-style transactional structures.
